# LearnFlow User Guide

Welcome to the LearnFlow Platform - your AI-powered learning companion for Python programming.

## Getting Started

### Creating an Account

1. Visit [{{SITE_URL}}]({{SITE_URL}})
2. Click "Sign Up"
3. Enter your email and create a password
4. Verify your email address

### Dashboard Overview

Your dashboard shows:
- **Current Progress**: Your mastery level (0-100%)
- **Learning Streak**: consecutive days of activity
- **Current Module**: What you're learning now
- **Recent Activity**: Recent exercises and quizzes

## Taking Lessons

### Lesson Structure

Each lesson contains:
1. **Concept**: Explanation with examples
2. **Practice**: Interactive exercises
3. **Quiz**: Test your understanding
4. **Project**: Apply what you learned

### Using the Code Editor

The built-in code editor includes:
- **Syntax Highlighting**: Color-coded Python code
- **Auto-complete**: Smart suggestions as you type
- **Run Button**: Execute your code instantly
- **Output Panel**: See your results below the editor

```python
# Try this!
name = input("What's your name? ")
print(f"Hello, {name}!")
```

### Getting Help

If you're stuck:
1. Click the "Ask AI Tutor" button
2. Type your question
3. Get a personalized hint or explanation

## Mastery System

Your progress is tracked with a mastery score:

| Score | Level | Color | Description |
|-------|-------|-------|-------------|
| 0-40% | Beginner | Red | Foundational work needed |
| 41-70% | Learning | Yellow | Practice recommended |
| 71-90% | Proficient | Green | Refining skills |
| 91-100% | Mastered | Blue | Ready for advanced topics |
