# LearnFlow Student Guide

A comprehensive guide to mastering Python with LearnFlow.

## Learning Philosophy

LearnFlow uses AI-powered adaptive learning:
- **Personalized**: Content adapts to your pace
- **Interactive**: Write code, see results instantly
- **Supportive**: AI tutor helps when you're stuck
- **Motivating**: Track streaks and earn achievements

## Your First Day

### 1. Complete Onboarding

- Take the placement quiz (optional)
- Set your learning goals
- Choose your starting module

### 2. Try the Editor

```python
# Your first Python program
print("Hello, LearnFlow!")

# Try some math
print(2 + 2)

# Use variables
name = "Your Name"
print(f"Welcome, {name}!")
```

### 3. Complete Your First Exercise

1. Read the instructions
2. Write your solution in the editor
3. Click "Run" to test
4. Submit when ready

### Using the AI Tutor

The AI tutor can help you:
- **Explain Concepts**: "What is a loop?"
- **Debug Code**: "Why is my code not working?"
- **Suggest Improvements**: "How can I make this better?"
- **Provide Examples**: "Show me another example"

## Learning Strategies

### Active Learning

Don't just read - practice:
1. Try the examples yourself
2. Modify them to see what happens
3. Break things and fix them
4. Build your own variations

### When You're Stuck

1. **Re-read** the concept explanation
2. **Review** similar examples
3. **Ask the AI Tutor** for a hint
4. **Take a break** and come back later

### Effective Practice

- **Daily sessions** of 15-30 minutes are better than long sessions once a week
- **Focus on understanding**, not just completing exercises
- **Review previous topics** regularly
- **Teach others** to reinforce your learning

## Mastery Tracking

Your mastery level reflects true understanding:

### How It's Calculated
- **Quiz Scores**: 40% weight
- **Exercise Completion**: 30% weight
- **Code Quality**: 20% weight
- **Time & Attempts**: 10% weight

### Interpreting Your Score

| Your Action | Effect on Mastery |
|-------------|-------------------|
| Complete exercise on first try | +5% |
| Complete after hints | +3% |
| Complete after multiple attempts | +1% |
| Quiz 100% score | +10% |
| Quiz 70-90% score | +5% |
| Help requested | No penalty |
| Skip exercise | -2% |

## Common Mistakes

### Syntax Errors

```python
# ❌ Missing parenthesis
print "Hello"

# ✅ Correct
print("Hello")
```

### Indentation Errors

```python
# ❌ Inconsistent indentation
if True:
  print("Yes")
    print("Also yes")

# ✅ Correct (4 spaces)
if True:
    print("Yes")
    print("Also yes")
```

### Variable Naming

```python
# ❌ Vague names
x = 35
y = 65
z = x + y

# ✅ Descriptive names
hours_worked = 35
hours_overtime = 65
total_hours = hours_worked + hours_overtime
```
