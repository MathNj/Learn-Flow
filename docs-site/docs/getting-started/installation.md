# Installation

This guide will help you install and set up {{SITE_NAME}}.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** >= 18.0
- **npm** or **yarn**
- **Python** >= 3.10 (for certain scripts)

## Install Dependencies

```bash title="Install using npm"
npm install
```

```bash title="Install using yarn"
yarn install
```

## Verify Installation

```bash title="Start the development server"
npm run start
```

Navigate to `http://localhost:3000` to see the documentation site.
